### Installation
1. Clone this project.
   
    ```sh
    $ git clone https://github.com/skyapps-id/Express-Typescript-Mysql.git 
    ```

2. Install Dependency
   
    ```sh
    $ npm install 
    ```

3. Run Project
   
    ```sh
    Depeloyment Mode :
    $ npm run build
    $ npm run start

    Developer Mode :
    $ npm run dev 
    ```
